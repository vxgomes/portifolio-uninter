Ao longo do tempo, fui adquirindo mais experiência e, em um ano, fui promovido para Suporte Nível 2, o que me proporcionou a oportunidade de lidar com questões mais complexas e desenvolver uma visão mais estratégica de soluções tecnológicas. Durante esse período, aprendi a lidar com desafios diários, além de aprimorar minhas habilidades de comunicação e resolução de problemas.
Após minha experiência na Microdata, fui convidado a integrar a equipe da Thomson Reuters como Suporte Técnico. Aqui, pude aprofundar ainda mais meus conhecimentos e trabalhar com uma variedade de ferramentas e sistemas mais avançados, atendendo a uma base de clientes maior e mais diversificada.
Atualmente, estou me formando em Análise e Desenvolvimento de Sistemas pela Universidade Internacional de Curitiba (Uninter), onde estou adquirindo uma base sólida em programação, banco de dados e desenvolvimento de soluções tecnológicas. Acredito que a combinação da experiência prática e a formação acadêmica me proporciona uma visão ampla e atualizada sobre o mercado de TI, permitindo-me atuar com mais eficiência e inovação.
Sou uma pessoa que busca sempre novos desafios, estou em constante aprendizado e adoro trabalhar em equipe para criar soluções criativas e eficazes. Meu objetivo é continuar crescendo na área de tecnologia, buscando cada vez mais me especializar e contribuir para o desenvolvimento de soluções inovadoras.

Calculadora - Java Script

Esta é uma calculadora simples desenvolvida utilizando HTML, CSS e JavaScript, com o intuito de oferecer uma interface amigável e funcional para realizar operações matemáticas básicas como adição, subtração, multiplicação e divisão.

HTML: A estrutura da calculadora é definida com HTML, criando os botões numéricos e de operações, além da área onde o resultado é exibido. A interface contém uma caixa de entrada que mostra os números e operações à medida que são inseridos.
CSS: O estilo visual é criado com CSS, permitindo um layout simples e intuitivo. O design inclui botões de tamanho uniforme, cores suaves e um layout responsivo para garantir boa visualização em diferentes dispositivos. O foco é na clareza, com cores contrastantes para distinguir entre números e operações.br
JavaScript: A lógica da calculadora é implementada em JavaScript, permitindo a execução das operações matemáticas conforme o usuário interage com os botões. O código captura os valores inseridos, realiza as operações e exibe os resultados dinamicamente sem a necessidade de recarregar a página.

Relogio - Java Script

Este relógio simples foi desenvolvido utilizando HTML, CSS e JavaScript com o objetivo de exibir a hora atual de forma clara e precisa. O relógio mostra as horas, minutos e segundos, sendo atualizado automaticamente a cada segundo.

HTML: A estrutura do relógio é montada com HTML, criando uma área para exibir a hora atual. A estrutura consiste basicamente em um elemento que apresenta a hora no formato HH:MM:SS, facilitando a visualização para o usuário.
CSS: O design visual do relógio é elaborado com CSS, permitindo um layout minimalista e elegante. O relógio é centralizado na tela, com fontes grandes e claras, e pode incluir elementos como uma borda ou uma sombra suave para dar destaque. A aparência pode ser personalizada, alterando cores, fontes e tamanhos.
JavaScript: A lógica do relógio é implementada com JavaScript. O código utiliza a função setInterval para atualizar o relógio a cada segundo, obtendo a hora atual através do objeto Date() e formatando-a para exibição. Essa abordagem garante que o relógio sempre mostre a hora correta em tempo real.

Slideshow - Java Script

Este Slideshow simples foi desenvolvido utilizando HTML, CSS e JavaScript, com o objetivo de criar uma apresentação de imagens interativa e visualmente atraente. Ele permite que o usuário visualize uma sequência de imagens com transições suaves entre elas, proporcionando uma experiência visual fluida.

HTML: A estrutura do slideshow é definida em HTML, criando uma área para as imagens e os controles de navegação (botões de "próximo" e "anterior"). Cada imagem é embutida em uma estrutura de contêiner, garantindo que a exibição seja organizada.
CSS: O estilo visual do slideshow é aplicado com CSS, oferecendo um layout limpo e elegante. As imagens são exibidas uma de cada vez, com transições suaves para a próxima ou anterior. O uso de efeitos como desvanecimento e animações cria uma experiência visual interessante. O design é responsivo, ajustando-se bem em diferentes dispositivos, como desktops e celulares.
JavaScript: A lógica do slideshow é implementada com JavaScript. O código controla a navegação entre as imagens e a transição entre elas. Utiliza funções como setInterval() para avançar automaticamente as imagens após um intervalo de tempo e também escuta os eventos de clique nos botões de navegação para permitir que o usuário altere manualmente as imagens.
